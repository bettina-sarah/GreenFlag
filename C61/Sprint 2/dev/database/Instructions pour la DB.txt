
Dans un terminal dans le dossier database du projet faire la commande 
pip install psycopg3

puis apres l'installation faire la commande
"C:\Program Files\PostgreSQL\15\bin\pg_restore" --host=localhost --port=5432 --username=postgres --dbname=greenflag ".\GreenFlag.dump"

lorsqu'on a fini pour sauvegarder la db
"C:\Program Files\PostgreSQL\15\bin\pg_dump" --host=localhost --port=5432 --username=postgres --dbname=greenflag ".\GreenFlag.dump"
pg_dump -U your_username -h localhost -p 5432 --dbname=your_database_name --file=output_file.dump --format=c
